Installation

Up and running
in 4 steps.

Python 3.12+ required. Windows primary, Linux/macOS functional.

01
Clone the repository
git clone https://github.com/FastXStudios/CodeToolsPP.git cd CodeTools-
02
Create virtual environment
# Windows python -m venv .venv .\.venv\Scripts\Activate.ps1 # Linux / macOS python -m venv .venv source .venv/bin/activate
03
Install dependencies
pip install -r requirements.txt
04
Launch Code Tools++
python main.py
